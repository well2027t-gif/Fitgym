# Hospedagem do Fitgym

O aplicativo **Fitgym** está hospedado gratuitamente no **GitHub Pages** e pode ser acessado pelo link abaixo:

| Item | Valor |
| --- | --- |
| URL pública do aplicativo | https://well2027t-gif.github.io/Fitgym/ |
| Plataforma de hospedagem | GitHub Pages |
| Tipo de hospedagem | Gratuita |
| Repositório | https://github.com/well2027t-gif/Fitgym |

Este arquivo foi criado para deixar registrado, dentro do próprio projeto, o endereço oficial em que a aplicação está publicada.

## Observação importante

A versão publicada foi ajustada para funcionar em **hospedagem gratuita estática**. Por isso, os recursos que dependiam de backend dedicado e autenticação do servidor não foram mantidos nesta publicação específica. O núcleo principal do aplicativo, baseado em armazenamento local no navegador, permanece acessível pelo link acima.
